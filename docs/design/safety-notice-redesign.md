> 最后更新：2026-07-09
> 状态：设计确认中

## 页面故事（Scene）

> 校园林荫道公告栏。
>
> 一块深色原木板，表面有自然的木纹和轻微的磨损痕迹。
>
> 板上用铜钉钉着几张羊皮纸——管理员刚贴上去的安全提醒。
>
> 纸张大小不一，有的边角微微翘起，被风吹得轻轻卷边。
>
> 学生路过，停下脚步，扫一眼公告栏，几秒钟就知道今天哪些地方要小心。

**核心情绪：** 潮流、年轻、手工质感、轻松有趣、动漫感。

---

## 页面视觉设计理念

**关键词：** 木质公告栏 × 羊皮纸 × 铜钉 × 潮流动漫感

| 原则 | 说明 |
|------|------|
| 材质真实 | 木纹、羊皮纸纹理、铜钉——每个元素都有物理世界的质感 |
| 层次分明 | 页面背景 → 公告栏 → 纸张 → 文字/图钉，四层空间关系清晰 |
| 不规则之美 | 纸张有轻微旋转（±1°~3°），打破网格的死板 |
| 潮流感 | 不是"做旧怀旧"，而是"干净的材质 + 有趣的排列 + 微动效" |
| 克制的动效 | hover 时纸张轻微抬起，像风吹动——自然不花哨 |
| 现代骨架 | 底层是现代响应式网格，材质是装饰层，不干扰信息阅读 |

**一句话：** 现代 Web 布局 + 手工材质 + 潮流动漫气质 = 校园公告栏体验。

## 材质设计

### 公告栏 — 三层木纹板

| 属性 | 值 |
|------|-----|
| 深木 | `#69523A` — 边框、阴影、木纹暗部 |
| 基底 | `#846B4B` — 公告栏主体色 |
| 高光 | `#9A8663` — 木纹亮线 |
| 纹理 | 三层叠加：木纹条纹 + 高光线 + 深→基底→浅渐变 |
| 表面 | 哑光，不反光 |

**边框：**

| 属性 | 值 |
|------|-----|
| 有无 | 有——原木边框条 |
| 材质 | 比板面略深一度的木色 `#4A2F22` |
| 宽度 | `8px ~ 12px` |
| 角落 | 45° 斜切拼接 |
| 作用 | 让公告栏有"边界感" |

**阴影：**

| 元素 | 阴影 |
|------|------|
| 公告栏整体 | `0 8px 40px rgba(0,0,0,0.2)` — 木板有厚度，放在浅色背景上 |
| 边框内侧 | `inset 0 2px 8px rgba(0,0,0,0.15)` — 木板有厚度感 |

**尺寸：**

| 属性 | 值 |
|------|-----|
| 最大宽度 | `900px` |
| 最小高度 | `400px`（空状态时也有存在感） |
| 内边距 | `28px 32px` |
| 圆角 | `4px` |

**木纹 CSS 思路：**
```css
background:
  repeating-linear-gradient(
    92deg,
    transparent,
    transparent 8px,
    rgba(0,0,0,0.06) 8px,
    rgba(0,0,0,0.06) 9px
  ),
  linear-gradient(180deg, #6B4C3B, #5C3D2E, #4A2F22);
```

### 公告栏角落装饰 — 猫爪印

**位置：**

| 角落 | 数量 | 说明 |
|------|------|------|
| 左上角 | 1 个 | 主爪印，最明显 |
| 右下角 | 1 个 | 次爪印，稍小 |
| 其他角落 | 不放 | 两个足够，多了乱 |

**形态：**

| 属性 | 值 |
|------|-----|
| 类型 | 猫爪印（肉垫 + 4 个小趾印） |
| 风格 | 手绘/墨迹风格——像猫踩了墨水踩上去的 |
| 不是 | 写实猫爪、卡通猫爪、emoji |
| 而是 | 简笔画风格的爪印，像用毛笔随手画的 |

**尺寸与透明度：**

| 属性 | 左上角 | 右下角 |
|------|--------|--------|
| 大小 | `28px ~ 32px` | `20px ~ 24px` |
| 旋转 | `-15°` | `+25°` |
| 透明度 | `0.12` | `0.08` |
| 颜色 | `var(--color-forest-ink)` `#2D3436` | 同左 |

**感觉：** 猫爪印几乎要融进木纹里——不仔细看看不出来，仔细看发现了会觉得"有点意思"。是设计师埋的小彩蛋。

**实现思路：**
- 伪元素 `::before` / `::after` 在公告栏容器上
- CSS `background-image` 使用 inline SVG 的爪印形状
- `opacity` 极低，和木纹融为一体

### 纸张 — 羊皮纸

| 属性 | 值 |
|------|-----|
| 主色 | `#D3C6A8` — 羊皮纸基底 |
| 做旧 | `#C0B08D` — 纸张边缘、纹理 |
| 暗边 | `#A48E68` — 纸张最外缘 |
| Hover | `#DDD0B2` — Hover 时纸张变亮 |
| 纹理 | 径向渐变（中心亮边缘暗）+ 做旧色斑（伪元素） |
| 边缘 | 毛边 clip-path + 微卷角 |
| 质感 | 哑光、有厚度感、局部色斑 |
| 背面可见 | 底部边缘露出 3px 深色条 |

### 铜钉

| 属性 | 值 |
|------|-----|
| 形状 | 圆形，直径 `14px`，半球形 |
| 颜色 | 做旧铜 `#B87333`，中心略亮 `#D4944A`，边缘略暗 `#8B5E2A` |
| 质感 | 哑光金属，不反光——有岁月感的铜 |
| 阴影 | `1px 2px 4px rgba(0,0,0,0.35)` |
| 位置 | 纸张顶部中央，偏移 `±8px`（不完全居中） |
| 数量 | 每张纸 1 个 |

**偏移规律：** 预设值循环，和旋转角度配对：`[-6px, +4px, -2px, +8px, -4px, +2px]`

## 公告摆放

### 排列方式：网格 + 轻微旋转

```
┌─────────────────────────────────────┐
│                                     │
│   ┌─────────┐    ┌─────────┐       │
│   │  🔩 -2° │    │  🔩 +1° │       │
│   │  纸张1   │    │  纸张2   │       │
│   └─────────┘    └─────────┘       │
│                                     │
│      ┌─────────┐    ┌─────────┐    │
│      │  🔩 +2° │    │  🔩 -1° │    │
│      │  纸张3   │    │  纸张4   │    │
│      └─────────┘    └─────────┘    │
│                                     │
└─────────────────────────────────────┘
```

### 旋转规则

| 规则 | 值 |
|------|-----|
| 旋转范围 | `-3°` ~ `+4°` |
| 分配方式 | 预设 6 个旋转值，按索引循环：`[-3°, +2.5°, -1.5°, +4°, -2°, +3.5°]` |
| 为什么不用随机 | 随机值每次刷新都变，用户会觉得"纸在飘"——预设值稳定 |

### 重叠规则

| 规则 | 说明 |
|------|------|
| 是否允许重叠 | 不允许——每张纸完全可见 |
| 间距 | `18px ~ 22px`（`gap: 20px`） |
| 原因 | 重叠会遮挡信息，安全提醒不能被藏起来 |

### 网格

| 属性 | 值 |
|------|-----|
| 布局 | CSS Grid `auto-fill, minmax(260px, 1fr)` |
| 每行数量 | 桌面 2~3 张，平板 2 张，手机 1 张 |
| 对齐 | `align-items: start`（顶部对齐，纸张高度可以不同） |

## 光影设计

### 光源方向

统一光源：左上方向（约 135°），模拟下午斜射阳光。

### 层级阴影

| 元素 | 阴影 | 意义 |
|------|------|------|
| 公告栏 | `0 8px 40px rgba(0,0,0,0.2)` | 木板有厚度，放在浅色背景上 |
| 纸张（默认） | `2px 3px 8px rgba(0,0,0,0.12)` | 纸张贴在木板上，有轻微厚度 |
| 纸张（hover） | `4px 8px 20px rgba(0,0,0,0.2)` + `translateY(-4px)` | 纸张被风掀起，阴影拉大 |
| 铜钉 | `1px 2px 4px rgba(0,0,0,0.35)` | 铜钉有立体凸起感 |

### 光影层次（从后到前）

```
第1层：页面背景 (cream-paper)      — 柔和漫射光
第2层：公告栏木板                   — 自身有光影纹理
第3层：纸张                        — 纸张阴影投在木板上
第4层：文字/铜钉                   — 在纸张平面上
```

## 页面层级

```
Z轴层级（从低到高）：

z-0  页面背景 (cream-paper)
 │
z-1  公告栏容器（木纹板 + 原木边框，整体阴影）
 │
z-2  纸张（每张独立阴影，独立旋转）
 │    └── 纸张纹理层（伪元素 ::before）
 │
z-3  内容层
 │    ├── LocationBadge（地点标签）
 │    ├── 标题
 │    ├── 正文
 │    └── 日期
 │
z-4  铜钉（溢出纸张边缘，最高层）
 │
z-5  交互态
      └── hover 时纸张 translateY(-4px)，阴影加深
```

### 信息层级

| 优先级 | 元素 | 视觉处理 |
|--------|------|---------|
| 1 | 地点 + 标题 | 粗体、大字号、左对齐 |
| 2 | 正文 | 常规字重、中等行高 |
| 3 | 发布时间 | 小字号、灰色、底部 |

## Hover 交互

### 纸张 Hover

| 属性 | 默认值 | Hover 值 | 过渡 |
|------|--------|----------|------|
| `transform` | `rotate(预设值)` | `rotate(预设值) translateY(-4px)` | `0.3s ease` |
| `box-shadow` | `2px 3px 8px rgba(0,0,0,0.12)` | `4px 8px 20px rgba(0,0,0,0.2)` | `0.3s ease` |
| `z-index` | 自动 | 提升到最顶层 | 瞬间 |

**感觉：** 用户鼠标移到一张纸上，纸张轻轻抬起 4px，阴影变深变大——像风吹起了纸的一角。不旋转，不缩放，只是轻轻抬起。

### 铜钉 Hover

铜钉跟随纸张一起移动，不需要独立动画。

## 滚动体验

| 元素 | 行为 |
|------|------|

## Notice Poster — 单张公告设计规范

### 纸张材质

#### 颜色

| 区域 | 色值 | 说明 |
|------|------|------|
| 基底色 | `#F0D9A0` | 羊皮纸暖黄 |
| 中心亮部 | `#F5E2B8` | 微妙径向渐变，中心比边缘亮 |
| 边缘暗部 | `#E0C488` | 四周略深，做旧感 |
| 局部色斑 | `rgba(180, 140, 80, 0.08)` | 随机位置淡色斑，模拟羊皮纸不均匀 |

#### 纹理

| 属性 | 描述 |
|------|------|
| 表面质感 | 哑光，不反光——羊皮纸不会有光泽 |
| 颗粒感 | 微妙 noise 纹理，模拟皮革/纸张纤维 |
| 斑驳 | 局部有轻微明暗变化，不是均匀纯色 |
| 手感想象 | 摸上去有轻微凹凸感，不是光滑打印纸 |

#### 厚度

| 属性 | 值 |
|------|-----|
| 视觉厚度 | 底部边缘露出 2~3px 深色条 `#C4A870` |
| 意义 | 暗示纸张有物理厚度，不是一层薄纸 |
| 实现 | 伪元素，底部 2px 实色条 |

#### 边缘

| 属性 | 描述 |
|------|------|
| 类型 | 毛边——不规则、不锋利 |
| 感觉 | 像手撕的，不是裁纸刀切的 |
| 实现 | clip-path 多边形，每条边有 2~3 个微小偏移点 |
| 四边差异 | 每条边的毛边形状不同，不对称 |

#### 卷角

| 属性 | 描述 |
|------|------|
| 位置 | 右下角（固定位置，不随机） |
| 大小 | 约 12px x 12px 的三角形 |
| 颜色 | 卷起部分露出纸张背面——比正面略深 `#D8C090` |
| 阴影 | 卷角处有微妙内阴影，暗示纸张弯曲 |
| 感觉 | 像纸张右下角被风吹卷了 |

#### 做旧程度

| 等级 | 描述 |
|------|------|
| 程度 | 轻度做旧 |
| 不是 | 破烂、发霉、撕裂、污渍 |
| 而是 | 边缘微微泛黄、表面有轻微颗粒感、局部色斑 |
| 比喻 | 像图书馆里保存得当的老文档——有年份，但干净整洁 |

#### 阴影

| 状态 | 阴影值 |
|------|--------|
| 默认 | `2px 3px 8px rgba(0,0,0,0.12)` |
| Hover | `4px 8px 20px rgba(0,0,0,0.2)` |
| 方向 | 右下方（统一光源 135°） |

### 内容布局

#### 纸张比例

| 属性 | 值 |
|------|-----|
| 宽高比 | 无固定比例——高度由内容撑开 |
| 最小宽度 | `260px` |

### 随机旋转规则

#### 预设旋转值

| 序号 | 旋转角度 | 铜钉偏移 |

| 1 | `-2.0°` | `-6px` |

| 4 | `+2.0°` | `+8px` |
| 5 | `-1.5°` | `-4px` |
| 6 | `+0.5°` | `+2px` |

#### 分配规则

- 按纸张索引 `i % 6` 循环分配
- 不用 Math.random()——每次刷新角度不变
- 铜钉偏移和旋转角度配对，保证视觉自然

### Hover 规则

#### 纸张 Hover

| 属性 | 默认值 | Hover 值 | 过渡 |
|------|--------|----------|------|
| `translateY` | `0` | `-4px` | `0.3s ease` |
| `box-shadow` | `2px 3px 8px rgba(0,0,0,0.12)` | `4px 8px 20px rgba(0,0,0,0.2)` | `0.3s ease` |
| `z-index` | 自动 | 提升到最顶层 | 瞬间 |

**感觉：** 鼠标移到纸上，纸张轻轻抬起 4px，阴影变深变大。像风吹起了纸的一角。不旋转，不缩放，不弹跳——只是轻轻抬起。

**铜钉：** 跟随纸张一起移动，无独立动画。

**卷角：** 无变化，静态装饰。

## 设计总结

| 维度 | 决策 |
|------|------|
| 背景 | 页面 cream-paper，无砖墙 |

| Hover | 纸张抬起 5px + 阴影加深 + 纸张变亮，300ms |
| 滚动 | 正常滚动，无视差，无浮动 |
| 印章 | 无 |
| 图标 | 无，纯文字 |
| 配色 | 低饱和暖色系：木板棕 + 羊皮纸 + 深棕墨水 + 天空蓝点缀 |
| 猫爪印 | 公告栏角落两个，极低透明度，inline SVG |
